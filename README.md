The purpose of this project was to create a sushi restaurant website üç±. The project was developed for educational purposes with the help of video tutorials from YouTube, information from official and not-so-official sites on various

## ‚öôÔ∏è Functionality
The site is a sushi restaurant where you can choose and buy various products. The main functions are:
- product search
- filtering by category
- sort by price / popularity / alphabet
- the ability to view the number of calories in products
- adding products to the cart and managing their quantity
